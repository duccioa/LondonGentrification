The file WIP.py is the work in progress, used as a workflow
When steps sre defined and coded, they get moved to a specific file
The folder 'py_functions' contains functions that are called in the code
In the main folder, the codes are different steps of the data collection or analysis

*FoodAgencyPremises_AlchemyAPIquery.py*: set up and run a requst of context analysis    
*FoodAgencyPremises_exploratory.py*: exploraory analysis and graphs of the dataset from food.gov.uk     
*FoodAgencyPremises_getData.py*: fetch data from the website food.gov.uk     
*FoodAgencyPremises_parseXMLdata*: make a dataframe out of the downloaded data     
*FoodAgencyPremises_TokensFreqDist*: calculate the frequency of tokens from the BusinessNames
